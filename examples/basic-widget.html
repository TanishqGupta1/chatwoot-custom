Chatwoot Widget

This page mounts the Chatwoot web widget. Replace the placeholders below with your deployment details, then open this file in a browser (or serve it via a simple HTTP server).

  • baseUrl: URL of your Chatwoot instance (e.g. https://app.chatwoot.com or https://yourhost)
  • websiteToken: The token from the Inbox > Installation > Website channel

The script loads asynchronously; the widget appears as soon as it finishes.
